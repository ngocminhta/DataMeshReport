\section[Problem]{Setting up problem}
\frame{
    \frametitle{\textbf{The real-life problem}}
	\begin{block}{}
		Let's imagine that we have to send many packages to several cities domestically and internationally.
	\end{block}
	\pause
	Algorithmically, we need to solve two problems simultaneously (Traveling Salesman Problem and Bin Packing Problem). In particular:
	\pause
    \begin{itemize}
    	\item \textbf{State:} Any route of the ship and any numbers of used containers.
    	\pause
    	\item \textbf{Action:} Rearrange the packages from this container to another, and change the route of the ship.
    	\pause
    	\item \textbf{Goal:} Find the shortest route and the least number of used containers.
    	\pause
    	\item \textbf{Path cost:} Length of the route and number of containers.
    \end{itemize}
}
